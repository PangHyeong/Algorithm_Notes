## 📌 Study Rule

1. 문제를 먼저 충분히 고민한다.
2. 풀이 후 반드시 복기한다.
3. 시간 복잡도 / 공간 복잡도를 분석한다.
4. 같은 유형은 패턴화한다.
5. 주기적으로 다시 풀어본다.
